Introdução à HTML

Para curiosos e iniciantes


Prof. Camila Silva
Publicado em: 18/07/2025 1 visualizações

Sobre o Curso

Este curso é uma introdução completa ao HTML, a linguagem fundamental para criação de páginas web. Ideal para quem está começando no desenvolvimento front-end.


Público-alvo

Iniciantes em programação web

Nível

Básico

Duração

4 semanas (10 horas no total)

Formato

Vídeo aulas + exercícios práticos

O que você vai aprender

  • Estrutura básica de documentos HTML
  • Principais tags e elementos
  • Formulários e tabelas
  • Semântica HTML5
  • Boas práticas de desenvolvimento
